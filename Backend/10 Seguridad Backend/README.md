# Módulo 10 — Seguridad Backend

## Propósito

La seguridad no es un feature — es un requisito. Cada capa necesita protección: autenticación, autorización, defensa contra OWASP Top 10, y hardening.

## Temas

| Archivo | Tema | Conceptos clave |
|---------|------|-----------------|
| 10.1 | Autenticación y autorización | JWT, OAuth 2.0, OIDC, sessions, API keys, mTLS |
| 10.2 | OWASP Top 10 | SQLi, XSS, CSRF, SSRF, broken access control |
| 10.3 | Hardening | TLS config, secrets management, security headers, dependency scanning |

## Dependencias

```
Módulo 4 (TLS/Crypto) + Módulo 9 (Arquitectura) → Módulo 10
```

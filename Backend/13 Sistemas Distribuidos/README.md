# Módulo 13 — Sistemas Distribuidos

## Propósito

En producción, tu backend no es un servidor — es una red de sistemas que deben coordinarse. Este módulo cubre los fundamentos teóricos y prácticos: CAP, consenso, coordinación, mensajería distribuida, y patrones de escalabilidad.

## Temas

| Archivo | Tema | Conceptos clave |
|---------|------|-----------------|
| 13.1 | Fundamentos | CAP, PACELC, modelos de consistencia |
| 13.2 | Coordinación | Leader election, distributed locks, consensus |
| 13.3 | Algoritmos de consenso | Raft, Paxos, PBFT |
| 13.4 | Mensajería distribuida | Kafka, RabbitMQ, event sourcing, CQRS |
| 13.5 | Escalabilidad real | Horizontal scaling, consistent hashing, replication |
| 13.6 | Patrones | Saga, outbox, idempotency, exactly-once |

# Módulo 09 — Arquitectura Backend

## Propósito

La arquitectura es el trade-off entre simplicidad y escalabilidad. Desde monolitos hasta microservicios, cada estilo tiene su lugar. Este módulo cubre estilos arquitectónicos, componentes de infraestructura, patrones de resiliencia, rate limiting, y API design.

## Temas

| Archivo | Tema | Conceptos clave |
|---------|------|-----------------|
| 9.1 | Estilos arquitectónicos | Monolito, modular monolith, microservicios, serverless |
| 9.2 | Componentes de infraestructura | API Gateway, service discovery, service mesh, message broker |
| 9.3 | Resiliencia | Circuit breaker, retry, timeout, bulkhead, fallback |
| 9.4 | Rate limiting | Token bucket, leaky bucket, sliding window |
| 9.5 | API Design | REST conventions, versioning, pagination, error handling |
